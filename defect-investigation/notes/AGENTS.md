# notes/ Index

User-only observations that require running the game live. Cannot be reproduced by scripts.
Read these when you need ground-truth cat data (actual defects/mutations/abilities seen in-game).

## in-game-observations/

Per-cat folders with a summary `.txt` file and supporting screenshots.
The `.sav` file snapshot at the time of observation is also stored here.
See `in-game-observations/AGENTS.md` for the full cat index and instructions.

- `ICON_LEGEND.txt`          — Maps in-game stat icons to stat names (STR/DEX/CON/INT/SPD/CHA/LCK).
- `CatName-Template-Folder/` — Blank template for adding a new cat observation folder.
- `steamcampaign01_20260424_191107.sav` — Save snapshot at time of in-game observations.
